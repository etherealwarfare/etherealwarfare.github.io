Pet Boarding Service Database Project

In the course Database Concepts (CGS 2545C), I used MySQL 8.0.35 to develop a database to store and query information on hypothetical clients, pets, and employees. The goal of this project was to design a structured database to handle client bookings, pet check-ins, employee assignments, and service tracking for a pet boarding business. The database helps ensure accurate record-keeping and quick access to information.

Skills Learned

  • MySQL 8.0.35
  • Relational database design
  • Query optimization and data management
